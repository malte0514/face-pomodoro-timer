# face-pomodoro-timer
顔認識で「離席」を検知して自動停止するポモドーロタイマー。Google Antigravity (Gemini 3 Pro) で作成。 / Face recognition Pomodoro timer that pauses when you leave.

Google Antigravity (Gemini 3 Pro) を使用して作成された、座りすぎ防止のためのWebアプリです。
デバイスのインカメラでユーザーの着席状況を監視し、離席中（カメラに顔が映っていない間）はタイマーのカウントを停止します。

## 特徴
* **実質着席時間の計測:** 離席中は時間が止まるため、「実際に25分座り続けた」タイミングで休憩を促します。
* **プライバシー重視:** 顔認識処理はすべてブラウザ内（MediaPipe）で行われます。映像がサーバーに送信されることはありません。
* **フルカスタマイズ:** フォント、音色、感度、作業時間を自由に設定可能。
* **PWA対応:** Androidタブレットやスマホのホーム画面に追加してアプリとして使用可能。

## 使い方
1.  URLにアクセスし、カメラの使用を許可してください。
2.  画面右下のタイマーの横に「●」が表示されていれば認識中です。
3.  25分経過するとアラームが鳴ります。ボタンを押して休憩してください。
4.  右上の歯車アイコンから設定を変更できます。
